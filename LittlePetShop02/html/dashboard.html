O gráfico exibido combina dados de diferentes períodos do dia (manhã, dia e noite) ao longo de uma semana. Cada dia da semana é representado no eixo horizontal, enquanto a quantidade de alguma medida (representada pela altura das barras) é mostrada no eixo vertical. Além das barras que representam as quantidades, uma linha é usada para indicar outra medida específica (representada pela cor da linha).
